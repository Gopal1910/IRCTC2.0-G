# 🚆 IRCTC 2.0 – Next-Generation Train Booking Platform

IRCTC 2.0 is a modern, fast, and user-centric redesign of the railway ticket booking platform.  
It delivers a seamless and intuitive experience with a smart search system, interactive seat selection, AI-powered travel assistance, real-time train tracking, and secure payments — all wrapped in a fully responsive and modern UI.

## 🌟 Key Highlights
- ⚡ Built with **React + TypeScript** for performance and reliability  
- ☁️ **Firebase Backend** for authentication, database & security  
- 🎯 Clean, responsive & user-friendly UI  
- 🚆 Train Booking + 🚈 Metro Ticket Booking + 🏨 Hotel Booking  
- 💺 Favourite Seat Selection & Seat Preference Memory  
- 👨‍🦰👩 Gender-Based Seat Availability Indicators  
- 🔎 Smart Search with Filters & Suggestions  
- 🔐 Secure and seamless booking experience  

---

## 🧠 Features

### 🔥 Core Booking Features
- Train ticket booking with smooth guided flow

- Hotel booking integration
